# Лабораторная работа №10-11. Создание приложения «Список студентов» с использованием Material Design

В ходе данной лабораторной работы мы на практике изучили возможности Matetial Design и сделали свое приложение

## Список вохможностей

- Отображение списка всех студентов
- Отображение Фио
- Отображение полных лет
- Возможность посмотреть описание студента


## Список Технологий

- Material Design 3
- Jetpack Compose
- Composable functions
- Spring Animation
